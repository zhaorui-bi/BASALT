# Welcome to BASALT: Binning Across a Series of Assemblies Toolkit!

<div class="result" markdown>

<!-- ![Image title](img/logo.png){ align=right width=300 } -->

This repository contains the documentation for **BASALT** (Binning Across a Series of Assemblies Toolkit), a comprehensive toolkit designed to enhance genome-resolved metagenomic analysis by refining binning processes across multiple assemblies. BASALT integrates various modules to improve the quality and completeness of metagenome-assembled genomes (MAGs).
